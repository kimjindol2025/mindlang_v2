# ℝ^n: ΠΛΗΡΗΣ ΓΕΩΜΕΤΡΙΚΗ ΘΕΜΕΛΙΩΣΗ

## I. Ορισμός του Χώρου Κατάστασης

Ζ := (ℝ^d, +, ·, ⟨·,·⟩)

Όπου:
- ℝ^d = σύνολο όλων των d-διάστατων διανυσμάτων στήλης
- + : ℝ^d × ℝ^d → ℝ^d (πρόσθεση διανυσμάτων)
- · : ℝ × ℝ^d → ℝ^d (βαθμωτός πολλαπλασιασμός)
- ⟨·,·⟩ : ℝ^d × ℝ^d → ℝ (εσωτερικό γινόμενο)

Αξιώματα:

1. Κλειστότητα: ∀u, v ∈ ℝ^d: u + v ∈ ℝ^d
2. Σύνδεση: u + (v + w) = (u + v) + w
3. Ουδέτερο: ∃0 ∈ ℝ^d: ∀v ∈ ℝ^d, v + 0 = v
4. Αντίστροφο: ∀v ∈ ℝ^d, ∃(-v): v + (-v) = 0
5. Κομμutativité: u + v = v + u
6. Κατανομή: α(u + v) = αu + αv, α ∈ ℝ
7. Συνδυασμός: α(βv) = (αβ)v
8. Ταυτότητα: 1 · v = v

Ορισμός εσωτερικού γινομένου:

⟨u, v⟩ := u^T v = Σᵢ₌₁^d uᵢvᵢ

Ιδιότητες:
- Συμμετρία: ⟨u, v⟩ = ⟨v, u⟩
- Διγραμμικότητα: ⟨αu + βv, w⟩ = α⟨u, w⟩ + β⟨v, w⟩
- Θετική Ορισμένοτητα: ⟨v, v⟩ ≥ 0, με ισότητα ⟺ v = 0

## II. Νόρμες και Μετρικές

Νόρμα L₂:
||v||₂ := √(⟨v, v⟩) = √(Σᵢ₌₁^d vᵢ²)

Ιδιότητες νόρμας:
1. ||v||₂ ≥ 0, με ισότητα ⟺ v = 0
2. ||αv||₂ = |α| · ||v||₂
3. ||u + v||₂ ≤ ||u||₂ + ||v||₂ (τριγωνική ανισότητα)

Άλλες νόρμες:
- ||v||₁ := Σᵢ₌₁^d |vᵢ|
- ||v||_∞ := maxᵢ |vᵢ|
- ||v||_p := (Σᵢ₌₁^d |vᵢ|^p)^(1/p), p ≥ 1

Μετρική Ευκλείδειας:
d(u, v) := ||u - v||₂

Ιδιότητες:
- d(u, v) ≥ 0
- d(u, v) = 0 ⟺ u = v
- d(u, v) = d(v, u)
- d(u, w) ≤ d(u, v) + d(v, w)

## III. Γραμμικές Μετασχηματισμοί

Ορισμός:
f: ℝ^m → ℝ^n είναι γραμμική ⟺ ∀u, v ∈ ℝ^m, α, β ∈ ℝ:
  f(αu + βv) = αf(u) + βf(v)

Μητρική Αναπαράσταση:
f(x) = Ax, όπου A ∈ ℝ^{n×m}

Ιδιότητες:
- Σύνθεση: (f ∘ g)(x) = f(g(x)) = AB x, όπου A ∈ ℝ^{p×n}, B ∈ ℝ^{n×m}
- Ταυτότητα: I_{n} x = x, I_{n} ∈ ℝ^{n×n}, (I)ᵢⱼ = δᵢⱼ
- Μηδέν: 0_{n×m} x = 0

Πίνακας Μετάθεσης:
A^T ∈ ℝ^{m×n} εάν A ∈ ℝ^{n×m}
(A^T)ᵢⱼ = Aⱼᵢ

Ιδιότητες:
- (A^T)^T = A
- (AB)^T = B^T A^T
- (A + B)^T = A^T + B^T

Αντίστροφος Πίνακας:
A^{-1} ∈ ℝ^{n×n} ⟺ AA^{-1} = A^{-1}A = I_n
Υπάρχει ⟺ det(A) ≠ 0

Ορθογώνιος Πίνακας:
Q ∈ ℝ^{n×n} είναι ορθογώνιος ⟺ Q^T Q = QQ^T = I_n
Ισοδύναμα: Q^{-1} = Q^T
Ιδιότητα: ||Qx||₂ = ||x||₂ (διατήρηση μήκους)

## IV. Ιδιόχρεων και Ιδιοδιανυσμάτων

Ορισμός:
λ ∈ ℝ είναι ιδιοχρέη του A ∈ ℝ^{n×n} ⟺ ∃v ≠ 0: Av = λv

Το v καλείται αντίστοιχο ιδιοδιάνυσμα.

Χαρακτηριστικό Πολυώνυμο:
det(A - λI) = 0

Ιδιότητες:
- Αν A ∈ ℝ^{n×n}, τότε το χαρακτηριστικό πολυώνυμο έχει βαθμό n
- Υπάρχουν (πιθανώς) n πραγματικές ή μιγαδικές ιδιοχρέες
- Αν A συμμετρική (A = A^T), όλες οι ιδιοχρέες είναι πραγματικές
- Αν A συμμετρική, τα αντίστοιχα ιδιοδιανύσματα είναι ορθογώνια

Φασματική Ανάλυση (Spectral Decomposition):
Εάν A ∈ ℝ^{n×n} συμμετρική με ιδιοχρέες λ₁ ≥ λ₂ ≥ ... ≥ λ_n και ορθonormα ιδιοδιανύσματα v₁, v₂, ..., v_n:

A = Σᵢ₌₁^n λᵢ vᵢ vᵢ^T

Ή σε μητρική μορφή:
A = VΛV^T, όπου V = [v₁ | v₂ | ... | v_n], Λ = diag(λ₁, λ₂, ..., λ_n)

## V. Παραγώγιση σε Πολλαπλών Μεταβλητών

Βαθμωτή Συνάρτηση:
f: ℝ^n → ℝ

Κλίση (Gradient):
∇f = (∂f/∂x₁, ∂f/∂x₂, ..., ∂f/∂x_n)^T ∈ ℝ^n

Ιδιότητες:
- ∇(αf + βg) = α∇f + β∇g
- ∇(fg) = f∇g + g∇f
- ∇(f/g) = (g∇f - f∇g)/g²
- ∇(||x||₂²) = 2x
- ∇(x^T Ax) = (A + A^T)x = 2Ax εάν A συμμετρική

Εσσιανή Μήτρα:
H_f = ∇²f ∈ ℝ^{n×n}
(H_f)ᵢⱼ = ∂²f/(∂xᵢ∂xⱼ)

Για συμμετρικές συναρτήσεις (δεύτερες μερικές παράγωγοι συνεχείς):
H_f συμμετρική: (H_f)ᵢⱼ = (H_f)ⱼᵢ

Ανάπτυγμα Taylor:
f(x + δx) ≈ f(x) + ∇f(x)^T δx + (1/2) δx^T H_f(x) δx

Ιακωβιανή Μήτρα:
f: ℝ^m → ℝ^n

J_f ∈ ℝ^{n×m}
(J_f)ᵢⱼ = ∂fᵢ/∂xⱼ

Όπου f = (f₁, f₂, ..., f_n)^T

## VI. Γραμμικές Δευτεροβάθμιες Μορφές

Δευτεροβάθμια Μορφή:
Q(x) := x^T Ax, όπου A ∈ ℝ^{n×n}

Χωρίς απώλειας γενικότητας, θεωρούμε A συμμετρική (αν όχι, αντικαθιστούμε με (A + A^T)/2).

Θετική Ορισμένοτητα:
- A ≽ 0 (θετικά ημιορισμένη) ⟺ ∀x ≠ 0: x^T Ax ≥ 0
- A ≻ 0 (θετικά ορισμένη) ⟺ ∀x ≠ 0: x^T Ax > 0

Κριτήρια:
- A ≻ 0 ⟺ όλες οι ιδιοχρέες λᵢ > 0
- A ≽ 0 ⟺ όλες οι ιδιοχρέες λᵢ ≥ 0
- A ≻ 0 ⟺ όλες οι κύριες ανήλικες (principal minors) > 0 (Sylvester's criterion)

Norma πίνακα:
||A||₂ := maxₓ≠₀ ||Ax||₂/||x||₂ = σ_max(A) (μεγαλύτερη singular value)

Spectral Norm ιδιότητες:
- ||AB||₂ ≤ ||A||₂ · ||B||₂
- ||Ax||₂ ≤ ||A||₂ · ||x||₂
- ||A^T A||₂ = ||A||₂²

## VII. Τανυστικές Δομές

Τανυστής: T ∈ ℝ^{d₁ × d₂ × ... × d_k}

Γραμμική Ανάπτυξη (Unfolding/Matricization):
T ∈ ℝ^{d₁ × d₂ × d₃} μπορεί να ξεδιπλωθεί σε:
T₍₁₎ ∈ ℝ^{d₁ × (d₂·d₃)} (mode-1 unfolding)
T₍₂₎ ∈ ℝ^{d₂ × (d₁·d₃)} (mode-2 unfolding)
T₍₃₎ ∈ ℝ^{d₃ × (d₁·d₂)} (mode-3 unfolding)

Τανυστικό Εσωτερικό Γινόμενο:
⟨T, S⟩ := Σᵢ₁₌₁^d₁ Σᵢ₂₌₁^d₂ ... Σᵢₖ₌₁^d_k Tᵢ₁,ᵢ₂,...,ᵢₖ · Sᵢ₁,ᵢ₂,...,ᵢₖ

Frobinius Norm:
||T||_F := √⟨T, T⟩

Τανυστικό Γινόμενο (Tensor Product):
u ⊗ v ∈ ℝ^{m×n}, όπου u ∈ ℝ^m, v ∈ ℝ^n
(u ⊗ v)ᵢⱼ = uᵢ · vⱼ

Ιδιότητες:
- (αu) ⊗ v = α(u ⊗ v)
- (u + v) ⊗ w = u ⊗ w + v ⊗ w
- (u ⊗ v) ⊗ w = u ⊗ (v ⊗ w) (ψευδο-συνδυασμός)

Kronecker Γινόμενο:
A ⊗ B ∈ ℝ^{mp×nq}, όπου A ∈ ℝ^{m×n}, B ∈ ℝ^{p×q}

A ⊗ B = [a₁₁B  a₁₂B  ...  a₁ₙB]
        [a₂₁B  a₂₂B  ...  a₂ₙB]
        [  ⋮    ⋮    ⋱   ⋮  ]
        [aₘ₁B  aₘ₂B  ...  aₘₙB]

Ιδιότητες:
- (A ⊗ B)(C ⊗ D) = (AC) ⊗ (BD)
- (A ⊗ B)^T = A^T ⊗ B^T
- vec(ABC) = (C^T ⊗ A) vec(B)

## VIII. Ανάλυση Ιδιαζουσών Τιμών (Singular Value Decomposition)

SVD Θεώρημα:
∀A ∈ ℝ^{m×n}, ∃U ∈ ℝ^{m×m}, Σ ∈ ℝ^{m×n}, V ∈ ℝ^{n×n} ορθογώνια:

A = UΣV^T

Όπου Σ = diag(σ₁, σ₂, ..., σᵣ, 0, ..., 0), σ₁ ≥ σ₂ ≥ ... ≥ σᵣ > 0, r = rank(A)

Γεωμετρική Ερμηνεία:
- V: ορθογώνια δεδομένα (principal axes στο χώρο εισόδου)
- Σ: κλιμακώσεις (singular values)
- U: ορθογώνια έξοδος (principal axes στο χώρο εξόδου)

Χρήσιμες Ιδιότητες:
- rank(A) = αριθμός μη-μηδενικών singular values
- ||A||₂ = σ₁ (μεγαλύτερη singular value)
- ||A||_F = √(Σᵢσᵢ²)
- Αν σᵣ > 0 και m = n, τότε A^{-1} = VΣ^{-1}U^T

Reduced SVD:
A ≈ U_r Σ_r V_r^T, όπου U_r ∈ ℝ^{m×r}, Σ_r ∈ ℝ^{r×r}, V_r ∈ ℝ^{n×r}

Eckart-Young Θεώρημα:
||A - A_k||_F = min_{rank(B)≤k} ||A - B||_F σημειώνεται όταν B = U_k Σ_k V_k^T

## IX. Χώροι Hilbert και Orthogonal Projections

Χώρος Hilbert: (ℝ^n, +, ·, ⟨·,·⟩) + πληρότητα

Μαθηματικά σημαντικό: κάθε Cauchy ακολουθία συγκλίνει.

Orthogonal Complement:
V^⊥ := {x ∈ ℝ^n | ∀v ∈ V: ⟨x, v⟩ = 0}

Ιδιότητες:
- V ⊕ V^⊥ = ℝ^n (ορθογώνια απευθείας άθροισμα)
- (V^⊥)^⊥ = V
- dim(V) + dim(V^⊥) = n

Orthogonal Projection:
P_V: ℝ^n → V, ό π ο ν P_V(x) = xₚᵣoⱼ ∈ V με x - P_V(x) ⊥ V

Αν V = span{v₁, v₂, ..., v_k} με orthonormal {vᵢ}:
P_V = Σᵢ₌₁^k vᵢ vᵢ^T

Ή σε μητρική μορφή με Q = [v₁ | v₂ | ... | v_k]:
P_V = QQ^T

Αν Q δεν έχει ορθonormal στήλες (Q^T Q ≠ I):
P_V = Q(Q^T Q)^{-1}Q^T

## X. Πολυγραμμικές Μορφές και Τανυστικές Αποσυνθέσεις

Multilinear Form:
f: ℝ^{n₁} × ℝ^{n₂} × ... × ℝ^{n_k} → ℝ
f γραμμική σε κάθε μεταβλητή.

Αναπαράσταση με τανυστή:
f(x⁽¹⁾, x⁽²⁾, ..., x⁽ᵏ⁾) = T ×₁ x⁽¹⁾ ×₂ x⁽²⁾ ... ×ₖ x⁽ᵏ⁾

Όπου ×ᵢ είναι mode-i tensor-vector contraction.

Mode-i Tensor-Vector Contraction:
(T ×ᵢ v)ᵢ₁,...,ᵢ_{ᵢ₋₁},ᵢ_{ᵢ₊₁},...,ᵢ_k = Σⱼ Tᵢ₁,...,ᵢ_{ᵢ₋₁},ⱼ,ᵢ_{ᵢ₊₁},...,ᵢ_k · vⱼ

CP Decomposition (PARAFAC):
T ≈ Σʳ₌₁^R aᵣ ⊗ bᵣ ⊗ cᵣ, όπου aᵣ ∈ ℝ^{n₁}, bᵣ ∈ ℝ^{n₂}, cᵣ ∈ ℝ^{n₃}

Tucker Decomposition:
T ≈ G ×₁ U⁽¹⁾ ×₂ U⁽²⁾ ×₃ U⁽³⁾, όπου G ∈ ℝ^{r₁×r₂×r₃}, U⁽ⁱ⁾ ∈ ℝ^{nᵢ×rᵢ}

## XI. Αναδρομές και Σταθερά Σημεία

Σταθερό Σημείο:
x* είναι σταθερό σημείο του f: ℝ^n → ℝ^n ⟺ f(x*) = x*

Banach Fixed-Point Theorem:
Αν f είναι contraction (∃k < 1: ||f(x) - f(y)||≤ k||x - y|| ∀x, y) σε πλήρη μετρικό χώρο, τότε:
- ∃! σταθερό σημείο x* ∈ ℝ^n
- ∀x₀ ∈ ℝ^n, η αναδρομή xₙ₊₁ = f(xₙ) συγκλίνει στο x*

Ρυθμός Σύγκλισης (Convergence Rate):
||xₙ₊₁ - x*|| ≤ k||xₙ - x*|| → γεωμετρική σύγκλιση με λόγο k

Newton-Raphson Iteration:
xₙ₊₁ = xₙ - H_f(xₙ)^{-1} ∇f(xₙ)

Τοπική σύγκλιση: αν H_f(x*) ≻ 0, τότε ∃δ > 0: ||x₀ - x*|| < δ ⟹ xₙ → x* με τετραγωνική σύγκλιση.

## XII. Ολοκληρώματα σε Πολλαπλά Διαστάσεις

Lebesgue Integral:
∫_{ℝⁿ} f(x) dx := sup{∫_{ℝⁿ} s(x) dx | s ≤ f, s απλή}

Fubini Theorem:
∫_{ℝᵐ⁺ⁿ} f(x, y) d(x,y) = ∫_{ℝᵐ} (∫_{ℝⁿ} f(x, y) dy) dx

Substitution Rule:
∫_B f(x) dx = ∫_A f(φ(u)) |det(J_φ(u))| du

Όπου φ: A → B διφορμοί, J_φ Ιακωβιανή.

Μέσος Όρος σε Σφαίρα:
avg_{||x||₂=r} f(x) = (1/|S^{n-1}|) ∫_{||u||₂=1} f(ru) dσ(u)

Όπου S^{n-1} = μοναδιαία σφαίρα, |S^{n-1}| = 2π^{n/2}/Γ(n/2)
